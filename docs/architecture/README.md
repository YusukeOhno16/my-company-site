## 概要

このプロジェクトは以下のようなアーキテクチャを採用しています：

- Next.jsのApp Routerを使用したフルスタックアプリケーション
- サーバーサイドレンダリング (SSR) とクライアントサイドレンダリングの併用
- API Routesを使用したバックエンド機能の実装
- Vercelへのデプロイメント
