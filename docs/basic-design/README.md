## 概要

このプロジェクトはNext.jsを使用した企業ウェブサイトであり、以下のような基本設計を採用しています：

- フロントエンド: Next.js (React)
- スタイリング: Tailwind CSS
- バリデーション: Zod, React Hook Form
- メール送信: Nodemailer
